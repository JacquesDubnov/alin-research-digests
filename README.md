# ALIN Research Digests

Lightweight repo for the ALIN Weekly Research Scanner (Claude Code scheduled trigger).

Contains:
- `known-citations.txt` — Index of existing citations from the ALIN knowledge base (author, title, journal, year, DOI). Used for deduplication.
- `digests/` — Weekly PubMed research digest files generated by the scheduled agent.

This repo exists because the main Obsidian vault (1.2 GB) is too large for remote agents to clone. New digest files are synced into the vault periodically.
